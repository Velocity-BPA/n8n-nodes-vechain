# Automate VeChain Blockchain Operations with Our New n8n Community Node

We're excited to announce that Velocity BPA has released a powerful new community node for n8n: n8n-nodes-vechain. This node brings comprehensive VeChain blockchain integration directly into your n8n workflows, enabling you to automate blockchain operations without writing a single line of code.

## The Challenge of Blockchain Integration

Integrating blockchain functionality into business workflows has traditionally required specialized development skills and deep technical knowledge. Whether you're tracking supply chain assets, managing digital tokens, or interacting with smart contracts on VeChain, the barrier to entry has been significant. Teams often need dedicated blockchain developers just to build basic integrations.

## Introducing n8n-nodes-vechain

Our new VeChain node eliminates these barriers by bringing blockchain operations into the familiar n8n environment. Now you can integrate VeChain's enterprise-grade blockchain capabilities into your existing automation workflows with simple drag-and-drop functionality.

## Key Features

The n8n-nodes-vechain node provides comprehensive VeChain functionality:

- **VET/VTHO Operations**: Monitor balances, transfer tokens, and track transactions for VeChain's native cryptocurrencies
- **VIP-180/181 Token Support**: Full support for VeChain's fungible and non-fungible token standards, enabling advanced asset management
- **Smart Contract Interactions**: Read from and write to VeChain smart contracts, opening up endless automation possibilities
- **Enterprise-Ready**: Built on VeChain's enterprise blockchain platform, designed for real-world business applications

## Real-World Use Cases

Imagine automatically updating your inventory system when VIP-181 NFTs are transferred, triggering notifications when wallet balances change, or creating automated supply chain workflows that interact with smart contracts. These scenarios and many more are now possible without custom development.

## Getting Started

Installation is simple. In your n8n instance, run:


npm install n8n-nodes-vechain


After restarting n8n, you'll find the VeChain node available in your node palette, ready to use in your workflows.
